# 参考资料入口

`docs/references/` 只保存理论、材料、公式和背景文献。

它不记录：

- 当前项目进度；
- validation 结果；
- 历史 stage 过程；
- outputs 数据说明。

参考资料可按主题理解：

- LNO327 / nickelate material background；
- Casimir / Lifshitz theory；
- finite-q response / Ward identity；
- BdG / superconducting response；
- reflection / conductivity conversion。

当前 PDF 文件保留在本目录。条目索引见 `papers.md`。
