# 虚频轴 Local Sheet Response 输出

本目录保存 Casimir 前置阶段的 local $q=0$ sheet-response 接口诊断。

保存的矩阵用于比较 normal-state $\sigma(i\omega)$ 与 BdG

$$
\Sigma_{\mathrm{SC}}(i\omega)
= \frac{K_{\mathrm{total}}(i\omega)}{\omega_{\mathrm{eV}}}, \qquad n \ge 1 .
$$

这些输出不是最终 Casimir 输入。$n=0$ Matsubara 项、SI sheet conductivity
归一化、非局域 $q_{\parallel}$ 响应以及产生 torque 的角向各向异性机制仍需正式物理方案。
