# 虚频轴超导响应输出

本目录保存 BdG superconducting response 在虚频轴上的诊断结果：

$$
\Sigma_{\mathrm{SC}}(i\omega_n)
= \frac{K_{\mathrm{total}}(i\omega_n)}{\omega_{\mathrm{eV}}}, \qquad n \ge 1 .
$$

其中 $K_{\mathrm{total}}=K_{\mathrm{dia}}-K_{\mathrm{para}}$。这些文件是当前
local-response Casimir 初级计算使用的 sigma-like response，但不是最终 optical
conductivity，也不处理 $n=0$。
