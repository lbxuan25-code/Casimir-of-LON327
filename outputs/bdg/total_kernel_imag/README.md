# BdG Total-Kernel Outputs

- `data/`: `.npz` scans of
  $K_{\mathrm{total}}(i\xi) = K_{\mathrm{para}}(i\xi) + K_{\mathrm{dia}}$.
- `figures/`: $C_4$-symmetry and component diagnostics for
  $K_{\mathrm{total}}(i\xi)$.

These outputs are imaginary-axis BdG electromagnetic kernel diagnostics. They
are not Casimir inputs yet and are not labeled as experimental optical
conductivities.
