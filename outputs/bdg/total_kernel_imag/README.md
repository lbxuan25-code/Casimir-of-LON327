# BdG total kernel 输出

- `data/`：$K_{\mathrm{total}}(i\xi) = K_{\mathrm{dia}} - K_{\mathrm{para}}(i\xi)$
  扫描的 `.npz` 数据。
- `figures/`：$K_{\mathrm{total}}(i\xi)$ 的 $C_4$ 对称性和分量诊断图。

这里采用 positive current-current bubble convention，并保留 BdG paramagnetic
Nambu `1/2` prefactor。输出是虚频轴 BdG electromagnetic kernel；它们不标记为
实验 optical conductivity。
