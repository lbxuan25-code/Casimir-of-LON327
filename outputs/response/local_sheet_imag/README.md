# Local Sheet Response Imaginary-Axis Outputs

This directory stores local $q=0$ sheet-response interface diagnostics for the
pre-Casimir stage.

The saved matrices compare normal-state $\sigma(i\omega)$ with BdG

$$
\Sigma_{\mathrm{SC}}(i\omega)
= \frac{K_{\mathrm{total}}(i\omega)}{\omega_{\mathrm{eV}}}, \qquad n \ge 1 .
$$

These outputs are not final Casimir inputs. The $n=0$ Matsubara term, SI sheet
conductivity normalization, nonlocal $q_{\parallel}$ response, and torque-producing
angular anisotropy mechanism remain unresolved.
