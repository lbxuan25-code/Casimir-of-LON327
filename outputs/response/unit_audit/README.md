# 响应单位审计输出

本目录保存 Casimir 前置阶段的单位约定审计结果。

当前采用中性 sheet-conductivity convention：

$$
\sigma_{\mathrm{model}}
\rightarrow
\sigma_{\mathrm{sheet}}^{\mathrm{SI}}
= \frac{e^2}{\hbar}\sigma_{\mathrm{model}}
\rightarrow
\sigma_{\mathrm{reflection}}
= \frac{\sigma_{\mathrm{sheet}}^{\mathrm{SI}}}{\sigma_0}.
$$

这些输出只审计单位转换；$n=0$、finite-$q_{\parallel}$ 和真实各向异性机制仍由其他接口处理。
