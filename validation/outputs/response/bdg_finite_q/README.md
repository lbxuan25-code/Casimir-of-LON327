- `bdg_finite_q_validation_summary.md`：中文结论和边界说明。
- `bdg_finite_q_validation_status.json`：机器可读 gate marker。
- `command.sh`：主要复现入口。

旧的 `stageSC_*.json` / `stageSC_*.md` 已归并到上述 summary/status，不再作为长期输出散落保存。validation scripts 仍保留，可按需重新生成原始 stage 输出。
